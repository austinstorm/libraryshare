What are invite codes?

If your library isn't open-invitation, people will need an invite code from an existing user to join the site. This way each new member will be vetted by a current member of the community. You can issue invite codes for your existing users to distribute on this page.

Number of Invite Codes to issue per user
 

What if I just want to invite?

If you want to invite a new member, use the Account Dropdown and click on "My Invites". All admins have unlimited invites. This screen is if you want to add a member manually (if you have a physical library, for instance).
